3-2. URDF와 Xacro로 로봇 만들기

Chapter 3 · 문제 2

과제 목표
URDF가 무엇이고 어떤 구조인지, 링크와 조인트가 뭔지 조사한다. 박스형 몸통에 바퀴 두 개를 단 간단한 로봇을 URDF로 만들고, 같은 로봇을 Xacro로도 만들어 RViz2에서 바퀴가 제대로 도는지 확인한다.

1. URDF란?

URDF(Unified Robot Description Format)는 ROS에서 로봇의 물리 구조·외형·기하 정보를 적어두는 XML 기반 파일이다. RViz나 Gazebo 같은 프로그램이 이 파일을 읽어서 로봇을 화면에 그리고 물리 연산을 한다.

  • <robot> : 최상위 태그, 로봇 이름 정의
  • <link> : 로봇의 독립 부품(몸통, 바퀴 등)
  • <joint> : 링크와 링크를 잇는 관절(움직이는 방식)
  • <material> : 색상·투명도 같은 재질

2. 링크(Link)와 조인트(Joint)

링크

로봇의 강체 부품 하나를 뜻한다. 각 링크는 자기만의 좌표계를 가진다.

  • <visual> : 눈에 보이는 외형과 색
  • <collision> : 물리 엔진이 충돌을 계산하는 영역
  • <inertial> : 질량과 관성 (물리 법칙 적용용)

조인트

부모 링크와 자식 링크를 연결하면서 어떻게 움직일지를 정한다. 위치(origin)는 부모 링크 좌표계 기준이다.

  • fixed : 완전히 고정
  • continuous : 축을 기준으로 무한 회전 (바퀴)
  • revolute : 각도 제한이 있는 회전 (로봇 팔)
  • prismatic : 직선으로 미끄러지는 관절

<axis>는 관절이 회전하거나 움직이는 기준 축을 정한다.

3. URDF 소스 코드

몸통(원기둥) 하나에 좌우 바퀴를 continuous 조인트로 붙였다. 바퀴가 로봇이 굴러갈 수 있는 방향으로 회전하도록 축과 각도를 맞췄다.

<?xml version="1.0"?>
<robot name="materials">

  <material name="blue">
    <color rgba="0 0 0.8 1"/>
  </material>

  <material name="white">
    <color rgba="1 1 1 1"/>
  </material>
  
  <material name="black">
    <color rgba="0 0 0 1"/>
  </material>

  <link name="base_link">
    <visual>
      <geometry>
        <cylinder length="1.0" radius="0.4"/>
      </geometry>
      <!-- [오타 수정] 순수 URDF는 글자 pi/2를 연산하지 못하므로 숫자로 변경 (pi/2 -> 1.570796) -->
      <origin xyz="0 0 0.1" rpy="0 1.570796 0"/>
      <material name="white"/>
    </visual>
    <inertial>
      <mass value="10"/>
      <inertia ixx="1e-3" ixy="0.0" ixz="0.0" iyy="1e-3" iyz="0.0" izz="1e-3"/>
    </inertial>
  </link>
    
  <link name="right_wheel">
    <visual>
      <geometry>
        <cylinder length="0.1" radius="0.5"/>
      </geometry>
      <!-- [오타 수정] 글자 pi/2를 숫자로 변경 (pi/2 -> 1.570796) -->
      <origin xyz="0 0 0" rpy="0 0 0"/>
      <material name="black"/>
    </visual>
    <inertial>
      <mass value="1"/>
      <inertia ixx="1e-3" ixy="0.0" ixz="0.0" iyy="1e-3" iyz="0.0" izz="1e-3"/>
    </inertial>
  </link>

  <link name="left_wheel">
    <visual>
      <geometry>
        <cylinder length="0.1" radius="0.5"/>
      </geometry>
      <!-- [오타 수정] 글자 -pi/2를 숫자로 변경 (-pi/2 -> -1.570796) -->
      <origin xyz="0 0 0" rpy="0 0 0"/>
      <material name="black"/>
    </visual>
    <inertial>
      <mass value="1"/>
      <inertia ixx="1e-3" ixy="0.0" ixz="0.0" iyy="1e-3" iyz="0.0" izz="1e-3"/>
    </inertial>
  </link>

  <joint name="right_wheel_joint" type="continuous">
    <parent link="base_link"/>
    <child link="right_wheel"/>
    <!-- [오타 수정] 글자 pi/2를 숫자로 변경 (pi/2 -> 1.570796) -->
    <origin xyz="0 0.5 0" rpy="1.570796 0 0"/>
    <axis xyz="0 0 1"/>
  </joint>

  <joint name="left_wheel_joint" type="continuous">
    <parent link="base_link"/>
    <child link="left_wheel"/>
    <!-- [오타 수정] 글자 -pi/2를 숫자로 변경 (-pi/2 -> -1.570796) -->
    <origin xyz="0 -0.5 0" rpy="-1.570796 0 0"/>
    <axis xyz="0 0 -1"/>
  </joint>
</robot>

4. Xacro란?

Xacro(XML Macros)는 순수 URDF의 코드 중복과 가독성 문제를 해결하는 매크로 도구다.

  • <xacro:property> : 상수를 위에서 한 번 선언하고 전체에서 재사용 → 치수 변경이 쉬움
  • <xacro:macro> : 반복되는 부품(바퀴 등)을 함수처럼 묶어서 짧게 생성
  • 수학 연산 가능 : ${body_high+width}처럼 사칙연산을 직접 씀

5. Xacro 소스 코드

바퀴 두 개를 매크로로 묶고 reflect 값(1/-1)으로 좌우를 자동 배치했다. URDF에서 4줄 넘게 쓰던 관절 코드를 호출 2줄로 줄였다.

<?xml version="1.0"?>
<robot name="materials" xmlns:xacro="http://www.ros.org/wiki/xacro">
  <xacro:property name="width" value="0.4" />       <!-- 로봇 몸통 반지름 -->
  <xacro:property name="length" value="1.0" />      <!-- 로봇 몸통 길이 -->
  <xacro:property name="wheel_length" value="0.1" />      <!-- 로봇 몸통 높이 -->
  <xacro:property name="body_high" value="0.1" />  <!-- 바퀴 지름 -->
  <xacro:property name="wheelrad" value="${body_high+width}" />  <!-- 바퀴 지름 -->

  <material name="blue">
    <color rgba="0 0 0.8 1"/>
  </material>

  <material name="white">
    <color rgba="1 1 1 1"/>
  </material>
  
  <material name="black">
    <color rgba="0 0 0 1"/>
  </material>
  <xacro:macro name="default_inertial" params="mass">
    <inertial>
      <mass value="${mass}" /> <!-- 입력받은 무게 설정 -->
      <!-- 관성 모멘트 회전 저항값 (시뮬레이션 가속도 계산용 기본값) -->
      <inertia ixx="1e-3" ixy="0.0" ixz="0.0" iyy="1e-3" iyz="0.0" izz="1e-3" />
    </inertial>
  </xacro:macro>
  <xacro:macro name="wheel" params="prefix reflect">
    <joint name="${prefix}_wheel_joint" type="continuous">
      <axis xyz="0 0 ${reflect*1}"> <!-- 회전 중심축을 Y축(0 1 0)으로 설정 -->
      <parent link="base_link"/>  <!-- 부모: 아래에서 만들 하부 베이스 -->
      <child link="${prefix}_wheel"/> <!-- 자식: 방금 만든 바퀴 링크 -->
      <!-- 부모 기준 바퀴 위치: reflect가 1이면 앞쪽, -1이면 뒤쪽에 배치됨 -->
      <origin xyz="0 ${reflect*(body_high+width)} 0" rpy="${reflect*1.570796} 0 0"/>
    </joint>
  </xacro:macro>

  <link name="base_link">
    <visual>
      <geometry>
        <cylinder length="${length}" radius="${width}"/>
      </geometry>
      <!-- [오타 수정] 순수 URDF는 글자 pi/2를 연산하지 못하므로 숫자로 변경 (pi/2 -> 1.570796) -->
      <origin xyz="0 0 ${body_high}" rpy="0 1.570796 0"/>
      <material name="white"/>
    </visual>
    <xacro:default_inertial mass="10"/>
  </link>
    
  <link name="right_wheel">
    <visual>
      <geometry>
        <cylinder length="${wheel_length}" radius="${wheelrad}"/>
      </geometry>
      <!-- [오타 수정] 글자 pi/2를 숫자로 변경 (pi/2 -> 1.570796) -->
      <origin xyz="0 0 0" rpy="0 0 0"/>
      <material name="black"/>
    </visual>
    <xacro:default_inertial mass="1"/>
  </link>

  <link name="left_wheel">
    <visual>
      <geometry>
        <cylinder length="${wheel_length}" radius="${wheelrad}"/>
      </geometry>
      <!-- [오타 수정] 글자 -pi/2를 숫자로 변경 (-pi/2 -> -1.570796) -->
      <origin xyz="0 0 0" rpy="0 0 0"/>
      <material name="black"/>
    </visual>
    <xacro:default_inertial mass="1"/>
  </link>

  <xacro:wheel prefix="right" reflect="1"/>
    <xacro:wheel prefix="left" reflect="-1"/>
</robot>
