1. 수행 목표 및 단계 명세

1.1 수행 목표

  • 게시자(Publisher)와 구독자(Subscriber)를 동시에 사용하는 폐쇄 루프 제어 시스템(Closed-Loop Control System)을 구현하여 로봇의 자율 거동을 제어합니다.

1.2 수행 단계 및 조건

  1. 제어 노드가 시작되면 turtlesim_node의 거북이 로봇은 임의의 방향으로 직진합니다.
  2. 제어 노드는 거북이가 발행하는 현재 위치 토픽(/turtle1/pose)을 실시간으로 구독합니다.
  3. 로봇이 외곽 벽면에 충돌하지 않도록 경계면에 도달하기 전 적절한 시점에 곡선을 그리며 방향을 바꾼 후 다시 직진하도록 제어 명령(/turtle1/cmd_vel)을 발행합니다.
  4. 하나의 노드가 서브스크라이버와 퍼블리셔 기능을 동시에 갖도록 설계합니다.

2. `turtle_move_control.py` 소스 코드 구현

  • 파일명: turtle_move_control.py
  • 위치: ~/sw_ws/src/my_robot_controller/my_robot_controller/turtle_move_control.py
import rclpy
from rclpy.node import Node
from geometry_msgs.msg import Twist
from turtlesim.msg import Pose

class TurtleMoveControl(Node):
    def __init__(self):
        super().__init__('turtle_move_control_node')
        self.publisher_ = self.create_publisher(Twist, 'turtle1/cmd_vel', 10)
        self.subscription = self.create_subscription(
            Pose,
            'turtle1/pose',
            self.pose_callback,
            10
        )
        self.subscription  # prevent unused variable warning

    def pose_callback(self, msg):
        twist_msg = Twist()
        # 중심 좌표(5.544445) 기준 오차가 4.5 미만일 때 (안전 영역) 직진 기동
        if (abs(msg.x - 5.544445) < 4.5) and (abs(msg.y - 5.544445) < 4.5):
            twist_msg.linear.x = 1.0
        # 경계 구역 접근 시 속도를 줄이고 회전각을 인가하여 충돌 회피 기동
        else:
            twist_msg.linear.x = 0.4  
            twist_msg.angular.z = 1.0  
        self.publisher_.publish(twist_msg)
        self.get_logger().info(f"Publishing: linear.x={twist_msg.linear.x}, angular.z={twist_msg.angular.z}")

def main(args=None):
    rclpy.init(args=args)
    turtle_move_control_node = TurtleMoveControl()

    try:
        rclpy.spin(turtle_move_control_node)
    except KeyboardInterrupt:
        turtle_move_control_node.get_logger().info("사용자에 의해 거북이 이동 제어 노드가 안전하게 종료됩니다.")
    finally:
        turtle_move_control_node.destroy_node()
        if rclpy.ok():
            rclpy.shutdown()

if __name__ == '__main__':
    main()

3. 실습 구동 및 결과 검증

  • 자체 개발한 제어 노드를 실행하여 거북이가 중심에서 출발한 뒤, 외곽 벽 영역에 근접하면 안전하게 선회하여 폐쇄된 사각 루프 경로 내에 갇혀 무한 궤도 기동을 유지하는 것을 확인했습니다.
